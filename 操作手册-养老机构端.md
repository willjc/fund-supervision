## 第一章 系统概述

### 1.1 系统简介

养老机构预收费资金监管平台养老机构端是面向养老机构工作人员的业务管理系统，主要功能包括：
- 机构信息维护与公示
- 床位资源管理
- 老人入住办理
- 订单与支付管理
- 费用申请与资金划拨
- 银行对账与流水查询

### 1.2 系统访问

**访问地址**: http://服务器地址/pension

**默认账号**: 由民政部门在审批通过后提供

## 第二章 登录与首页

### 2.1 系统登录

1. 打开浏览器，访问系统地址
2. 输入用户名和密码
3. 点击"登录"按钮
4. 首次登录后建议修改默认密码

### 2.2 首页数据看板

登录成功后，进入首页数据看板，展示以下内容：

#### 2.2.1 统计数据卡片

| 数据项 | 说明 |
|--------|------|
| 今日新增订单 | 当日新增订单数量 |
| 今日订单金额 | 当日订单总金额 |
| 在院老人数 | 当前在住的老人总数 |
| 账户余额汇总 | 所有老人账户余额总和 |

#### 2.2.2 图表分析

- **入住结构分析**: 性别比例、年龄分布、护理等级分布
- **床位使用率统计**: 已用床位、空置床位、维修床位
- **订单趋势图**: 近30天订单数量和金额趋势

## 第三章 机构管理

### 3.1 机构信息维护

#### 3.1.1 查看机构信息

1. 进入"机构管理"→"机构信息"
2. 查看机构基本信息：
   - 机构名称、统一信用代码
   - 法人代表、联系方式
   - 机构地址、床位数
   - 收费标准区间

#### 3.1.2 修改机构信息

1. 点击"编辑"按钮
2. 修改可变更的信息：
   - 联系人及电话
   - 机构地址
   - 服务特色
3. 点击"保存"提交
4. 修改后需重新提交民政部门审批

### 3.2 附件材料上传

#### 3.2.1 上传附件

1. 进入"机构管理"→"附件材料"
2. 查看已上传的附件列表
3. 点击"上传附件"按钮
4. 选择附件类型：
   - 营业执照
   - 消防验收合格证明
   - 食品经营许可证
   - 医疗执业许可证
   - 其他资质文件
5. 选择文件并上传
6. 点击"保存"提交

#### 3.2.2 附件要求

| 附件类型 | 格式要求 | 大小限制 |
|---------|---------|---------|
| 图片文件 | JPG/PNG/PDF | ≤ 5MB |
| 证照文件 | PDF/JPG | ≤ 10MB |

### 3.3 公示信息管理

#### 3.3.1 编辑公示信息

1. 进入"机构管理"→"公示信息"
2. 编辑以下内容：
   - 机构简介
   - 服务特色
   - 收费标准
   - 服务项目
3. 点击"发布"按钮

#### 3.3.2 机构图片管理

1. 进入"机构管理"→"机构图片"
2. 点击"上传图片"按钮
3. 选择图片类型：
   - 机构外观
   - 居住环境
   - 公共设施
   - 活动照片
4. 上传图片并添加说明
5. 点击"保存"提交

## 第四章 床位管理

### 4.1 床位列表

#### 4.1.1 查看床位列表

1. 进入"床位管理"→"床位列表"
2. 查看本机构所有床位信息
3. 可按以下条件筛选：
   - 楼层
   - 房间号
   - 床位状态（空置/占用/维修）

#### 4.1.2 床位状态说明

| 状态 | 说明 | 颜色标识 |
|------|------|---------|
| 空置 | 可分配给老人 | 绿色 |
| 占用 | 已有老人入住 | 红色 |
| 维修 | 床位维修中 | 灰色 |

### 4.2 新增床位

#### 4.2.1 单个新增

1. 点击"新增床位"按钮
2. 填写床位信息：
   - 楼层
   - 房间号
   - 床位号
   - 床位类型（普通/豪华/医疗）
   - 床位价格
   - 护理等级价格
3. 点击"保存"提交

#### 4.2.2 批量导入

1. 点击"批量导入"按钮
2. 下载导入模板
3. 按模板格式填写床位信息
4. 上传填写好的Excel文件
5. 系统自动校验并导入

### 4.3 餐费管理

#### 4.3.1 配置餐费标准

1. 进入"床位管理"→"餐费管理"
2. 点击"新增餐费标准"
3. 填写以下信息：
   - 餐费类型（早餐/午餐/晚餐）
   - 餐费标准
   - 适用护理等级
   - 备注说明
4. 点击"保存"提交

## 第五章 入住管理

### 5.1 新增入驻

#### 5.1.1 填写老人信息

1. 进入"入住管理"→"新增入驻"
2. 填写老人基本信息：
   - 姓名、性别、身份证号
   - 出生日期、联系电话
   - 家庭住址
   - 紧急联系人及电话
3. 上传身份证照片
4. 点击"下一步"

#### 5.1.2 填写健康信息

1. 填写健康状况：
   - 健康状况描述
   - 护理等级（自理/半护理/全护理）
   - 特殊需求说明
   - 既往病史
2. 点击"下一步"

#### 5.1.3 选择床位

1. 选择入住楼层
2. 选择房间号
3. 选择空置床位
4. 系统自动显示床位价格
5. 点击"下一步"

#### 5.1.4 确认订单

1. 确认订单信息：
   - 服务期限
   - 床位费用
   - 护理费用
   - 餐费标准
   - 押金金额
2. 点击"提交订单"

### 5.2 入住人列表

#### 5.2.1 查看入住人列表

1. 进入"入住管理"→"入住人列表"
2. 查看所有在院老人信息
3. 可按以下条件筛选：
   - 老人姓名（模糊搜索）
   - 入住状态
   - 护理等级
   - 楼层

#### 5.2.2 老人详情

点击老人姓名查看详情：
- 基本信息
- 健康状况
- 家庭成员
- 入住信息
- 账户余额
- 订单记录

### 5.3 老人信息管理

#### 5.3.1 修改老人信息

1. 在入住人列表中找到老人
2. 点击"编辑"按钮
3. 修改可变更的信息
4. 点击"保存"提交

#### 5.3.2 家庭成员管理

1. 进入老人详情页
2. 切换到"家庭成员"标签
3. 点击"添加家庭成员"按钮
4. 填写家庭成员信息：
   - 姓名、关系
   - 联系电话
   - 居住地址
5. 点击"保存"提交

### 5.4 续费办理

#### 5.4.1 创建续费订单

1. 在入住人列表中找到老人
2. 点击"续费"按钮
3. 选择续费月数
4. 确认费用明细
5. 点击"提交订单"

#### 5.4.2 续费订单支付

续费订单创建后，由老人家属通过H5端完成支付。

### 5.5 退费办理

#### 5.5.1 发起退费

1. 在入住人列表中找到老人
2. 点击"退费"按钮
3. 填写退费信息：
   - 退费原因
   - 退费类型（全额/部分）
   - 退费金额
   - 退费账户信息
4. 提交退费申请

#### 5.5.2 退费审批

退费申请需提交民政部门审批，审批通过后自动退款。

## 第六章 订单管理

### 6.1 订单列表

#### 6.1.1 查看订单列表

1. 进入"订单管理"→"订单列表"
2. 查看本机构所有订单
3. 可按以下条件筛选：
   - 订单号
   - 老人姓名
   - 订单状态
   - 下单时间

#### 6.1.2 订单状态说明

| 状态 | 说明 | 可执行操作 |
|------|------|----------|
| 审核通过待付款 | 待家属确认支付 | 催款 |
| 已支付 | 订单已支付完成 | 查看详情 |
| 已取消 | 订单已取消 | 查看取消原因 |
| 已退款 | 订单已退款 | 查看退款记录 |

### 6.2 订单详情

#### 6.2.1 查看订单详情

1. 点击订单号进入详情页
2. 查看以下信息：
   - 订单基本信息
   - 老人信息
   - 订单明细
   - 支付记录
   - 资金划拨计划

#### 6.2.2 订单明细

订单明细包含以下内容：
| 项目 | 说明 |
|------|------|
| 服务期限 | 订单服务的起止日期 |
| 床位费 | 床位月费金额 |
| 护理费 | 护理月费金额 |
| 餐费 | 每月餐费金额 |
| 会员费 | 一次性会员费 |
| 押金 | 可退还押金 |
| 订单总额 | 以上项目合计 |

### 6.3 订单支付

#### 6.3.1 线上支付

线上支付由老人家属通过H5端完成，机构端可查看支付状态。

#### 6.3.2 线下支付登记

如家属通过线下方式支付，需要登记支付记录：

1. 在订单列表中找到订单
2. 点击"线下支付"按钮
3. 填写支付信息：
   - 支付方式（现金/转账/支票）
   - 支付金额
   - 支付时间
   - 支付凭证号
4. 上传支付凭证照片
5. 点击"提交"完成登记

### 6.4 订单审核

#### 6.4.1 审核入驻订单

对于H5端提交的入驻订单，机构需要审核：

1. 进入"订单管理"→"待审核订单"
2. 查看待审核订单列表
3. 点击"审核"按钮
4. 确认订单信息无误
5. 点击"审核通过"或"审核驳回"

### 6.5 订单取消

#### 6.5.1 取消订单

1. 在订单列表中找到订单
2. 点击"取消"按钮
3. 选择取消原因
4. 填写取消说明
5. 点击"确定"提交

## 第七章 费用申请

### 7.1 押金使用申请

#### 7.1.1 发起押金使用申请

1. 进入"费用申请"→"押金使用申请"
2. 点击"新增申请"
3. 填写申请信息：
   - 选择老人
   - 申请金额
   - 申请原因
   - 使用事由
   - 紧急程度
4. 上传相关证明材料
5. 点击"提交"申请

#### 7.1.2 查看申请进度

1. 进入"费用申请"→"押金使用申请"
2. 查看申请列表及状态
3. 点击"查看详情"了解审批进度

#### 7.1.3 申请状态说明

| 状态 | 说明 | 可执行操作 |
|------|------|----------|
| 草稿 | 暂存未提交 | 编辑、提交 |
| 待家属审批 | 等待家属确认 | 撤回 |
| 家属已审批 | 家属已同意，待监管审批 | 无 |
| 待监管审批 | 等待民政部门审批 | 无 |
| 已通过 | 审批通过，资金已划拨 | 查看详情 |
| 已驳回 | 审批未通过 | 查看驳回原因 |
| 已撤回 | 申请已撤回 | 重新提交 |

### 7.2 资金划拨申请

#### 7.2.1 发起划拨申请

1. 进入"费用申请"→"资金划拨申请"
2. 点击"新增申请"
3. 填写申请信息：
   - 划拨金额
   - 划拨期间
   - 申请原因
   - 紧急程度
4. 点击"提交"申请

#### 7.2.2 查看划拨记录

1. 进入"费用申请"→"资金划拨记录"
2. 查看所有划拨记录
3. 可按时间范围筛选

## 第九章 常见问题

### 9.1 登录问题

**Q: 忘记登录密码怎么办？**

A: 联系民政部门系统管理员重置密码。

**Q: 登录后提示"机构未入驻"？**

A: 机构入驻申请可能还在审批中，或已被驳回，请联系民政部门确认。

### 9.2 床位问题

**Q: 床位占用状态与实际不符？**

A: 可能是老人已退住但未在系统中办理退住手续，请及时办理退住手续更新床位状态。

**Q: 如何批量修改床位价格？**

A: 目前暂不支持批量修改，需逐个编辑或联系技术人员处理。

### 9.3 订单问题

**Q: 订单创建后发现信息错误怎么办？**

A: 待支付状态的订单可以取消后重新创建；已支付订单请联系民政部门处理。

**Q: 家属支付后多久订单状态更新？**

A: 一般在支付成功后1-2分钟内更新，如超过5分钟未更新请联系技术支持。

### 9.4 费用问题

**Q: 押金使用申请多久能审批完成？**

A: 一般流程：家属审批（1天内）+ 监管审批（2-3个工作日），总计约3-5个工作日。

**Q: 资金划拨失败如何处理？**

A: 查看"资金划拨记录"中的失败原因，常见原因包括余额不足、账户信息错误等。核实后可重新发起申请。
