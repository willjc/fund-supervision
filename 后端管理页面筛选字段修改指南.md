# 后端管理页面筛选字段修改指南

## 一、修改目标

在后端管理系统的养老机构管理页面添加以下新字段的录入和展示:

1. 区县代码(district_code) - 下拉选择
2. 机构性质(institution_nature) - 单选
3. 收住类型(care_levels) - 多选
4. 医疗条件(medical_condition) - 单选
5. 星级评分(rating_level) - 数字选择
6. 价格区间(price_range_min, price_range_max) - 数字输入
7. 免费试住(free_trial) - 单选

## 二、涉及文件

- **列表页面**: `ruoyi-ui/src/views/pension/institution/index.vue`
- **表单对话框**: `ruoyi-ui/src/views/pension/institution/index.vue` (内嵌表单)
- **API接口**: `ruoyi-ui/src/api/pension/institution.js`

#### 2.5 修改表单提交逻辑

```vue
<script>
export default {
  methods: {
    // 提交表单
    submitForm() {
      this.$refs["form"].validate(valid => {
        if (valid) {
          // 处理收住类型数组转为逗号分隔字符串
          const submitData = {
            ...this.form,
            careLevels: this.form.careLevels.join(',')  // 数组转字符串
          }

          if (this.form.institutionId != null) {
            updateInstitution(submitData).then(response => {
              this.$modal.msgSuccess("修改成功")
              this.open = false
              this.getList()
            })
          } else {
            addInstitution(submitData).then(response => {
              this.$modal.msgSuccess("新增成功")
              this.open = false
              this.getList()
            })
          }
        }
      })
    },

    // 重置表单
    reset() {
      this.form = {
        institutionId: null,
        // ... 原有字段重置
        districtCode: undefined,
        institutionNature: undefined,
        careLevels: [],
        medicalCondition: undefined,
        ratingLevel: 3,  // 默认3星
        priceRangeMin: undefined,
        priceRangeMax: undefined,
        freeTrial: '0'
      }
      this.resetForm("form")
    },

    // 编辑机构
    handleUpdate(row) {
      this.reset()
      const institutionId = row.institutionId || this.ids[0]
      getInstitution(institutionId).then(response => {
        this.form = response.data
        // 将收住类型字符串转为数组
        if (this.form.careLevels) {
          this.form.careLevels = this.form.careLevels.split(',')
        } else {
          this.form.careLevels = []
        }
        this.open = true
        this.title = "修改养老机构"
      })
    },

    // 获取收住类型标签文本
    getCareLevelLabel(value) {
      const item = this.careLevelOptions.find(opt => opt.dictValue === value)
      return item ? item.dictLabel : value
    }
  }
}
</script>
```

#### 2.6 添加表单验证规则

```vue

export default {
```

## 六、常见问题

### 1. 字典数据加载失败
**原因**: SQL脚本未执行或字典类型名称错误
**解决**: 检查数据库字典数据,确认字典类型名称正确

### 2. 收住类型保存后变成 [object Object]
**原因**: 数组未转为字符串
**解决**: 提交前使用 `array.join(',')` 转换

### 3. 编辑时收住类型不显示
**原因**: 字符串未转为数组
**解决**: 回显时使用 `string.split(',')` 转换

### 4. 价格区间验证不生效
**原因**: 验证器未正确引用 form 对象
**解决**: 使用 `this.form` 访问另一个字段值

### 5. 星级评分无法选择
**原因**: 绑定值类型错误(字符串而非数字)
**解决**: 确保 ratingLevel 为 Number 类型

## 七、优化建议

1. **字典缓存**: 将字典数据缓存到 Vuex,避免每次加载页面都请求
2. **表单分步**: 如果字段过多,考虑使用 Steps 分步表单
3. **必填标识**: 在表单 label 前添加红色 * 标识必填项
4. **帮助提示**: 在复杂字段旁添加 Tooltip 帮助说明
5. **数据导出**: 在导出 Excel 时包含新增字段
