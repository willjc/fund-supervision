# 养老机构H5移动端项目搭建完整指南

## 📋 项目概述

**项目名称**: ruoyi-h5  
**技术栈**: Vue 2.6 + Vant 2.12 + Vuex + Vue Router + Axios  
**用户群体**: 老人、家属  
**核心功能**: 押金余额查询、家属审批押金使用申请、机构浏览、订单管理

---

## 🚀 第一步: 创建Vue 2项目

由于Vue CLI默认创建Vue 3项目,我们需要手动选择Vue 2:

```bash
cd D:\newhm\newzijin
vue create ruoyi-h5
```

**选择配置**:
- ❯ Manually select features
- 选择: Babel, Router, Vuex, CSS Pre-processors, Linter
- Vue版本: **2.x**
- Router history mode: **Yes**
- CSS预处理器: **Less**
- ESLint: **ESLint + Standard config**
- Lint on save: **Yes**
- 配置文件: **In dedicated config files**

---

## 📦 第二步: 安装依赖

```bash
cd ruoyi-h5

# 安装Vant 2.x
npm install vant@2.12.54 --save

# 安装核心工具库
npm install axios@0.27.2 js-cookie@3.0.5 dayjs@1.11.10 --save

# 安装移动端适配插件
npm install postcss-pxtorem@5.1.1 lib-flexible@0.3.2 --save

# 安装开发工具(可选)
npm install vconsole@3.15.1 --save-dev
```

## 📁 第四步: 创建目录结构

```bash
cd src

# 创建API目录
mkdir -p api

# 创建工具目录
mkdir -p utils config mixins

# 创建页面目录
mkdir -p views/home views/institution views/order views/user
mkdir -p views/account views/todo views/elder views/notice views/auth

# 创建组件目录
mkdir -p components/TabBar components/InstitutionCard components/OrderCard
```

**完整目录结构**:
```
src/
├── api/                    # API接口定义
│   ├── auth.js             # 认证接口
│   ├── institution.js      # 机构接口
│   ├── order.js            # 订单接口
│   ├── elder.js            # 老人档案接口
│   ├── account.js          # 费用账户接口(核心)
│   └── todo.js             # 待办事项接口
├── assets/                 # 静态资源
│   ├── images/             # 图片
│   └── styles/             # 全局样式
├── components/             # 公共组件
│   ├── TabBar/             # 底部导航
│   ├── InstitutionCard/    # 机构卡片
│   └── OrderCard/          # 订单卡片
├── config/                 # 配置文件
│   └── dict.js             # 数据字典✅
├── mixins/                 # 混入
│   └── listMixin.js        # 列表混入
├── router/                 # 路由配置
│   └── index.js            # 路由定义
├── store/                  # Vuex状态管理
│   ├── index.js            # Store入口
│   └── modules/            # 模块
│       ├── user.js         # 用户模块
│       └── elder.js        # 老人模块
├── utils/                  # 工具函数
│   ├── request.js          # axios封装✅
│   ├── auth.js             # Token管理✅
│   └── format.js           # 格式化工具✅
├── views/                  # 页面组件
│   ├── home/               # 首页
│   ├── institution/        # 机构
│   ├── order/              # 订单
│   ├── user/               # 我的
│   ├── account/            # 费用账户(核心)
│   └── auth/               # 登录
├── App.vue                 # 根组件
└── main.js                 # 入口文件
```
