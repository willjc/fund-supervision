## 第二章 登录与首页

### 2.1 系统登录

1. 打开浏览器，访问系统地址
2. 输入用户名和密码
3. 点击"登录"按钮
4. 首次登录后建议修改默认密码

### 2.2 首页数据看板

登录成功后，进入首页数据看板，展示以下内容：

#### 2.2.1 统计数据卡片

| 数据项 | 说明 |
|--------|------|
| 入驻机构数 | 已审批通过的养老机构数量 |
| 在院老人数 | 当前在住的老人总数 |
| 监管资金总额 | 所有机构监管账户余额总和 |
| 预警数量 | 当前未处理的预警数量 |

2. 选择"修改密码"

4. 点击"确定"保存

## 第三章 机构管理

### 3.1 机构批量导入

#### 3.1.1 下载导入模板

1. 进入"机构管理"→"机构列表"
2. 点击"批量导入"按钮
3. 下载Excel导入模板

#### 3.1.2 填写机构信息

按照模板格式填写以下信息：
- 机构名称
- 统一社会信用代码
- 机构地址
- 联系人及电话
- 床位数量
- 收费标准

#### 3.1.3 上传导入

1. 填写完成后保存Excel文件
2. 点击"选择文件"上传
3. 系统自动校验数据格式
4. 校验通过后自动创建机构和登录账号

### 3.2 机构入驻审批

#### 3.2.1 查看待审批机构

1. 进入"机构管理"→"机构审批"
2. 系统显示状态为"待审批"的机构列表
3. 可按申请时间、机构名称筛选

#### 3.2.2 审批机构详情

1. 点击机构名称进入详情页
2. 查看机构基本信息：
   - 机构名称、统一信用代码
   - 法人代表、联系方式
   - 机构地址、床位数
   - 收费标准区间
3. 查看附件材料：
   - 营业执照
   - 消防验收合格证明
   - 食品经营许可证
   - 三方协议

#### 3.2.3 审批操作

**审批通过**:
1. ���认机构信息完整、材料齐全
2. 点击"审批通过"按钮
3. 系统自动生成机构账号和密码
4. 机构状态变更为"已入驻"

**审批驳回**:
1. 点击"审批驳回"按钮
2. 填写驳回原因（必填）
3. 点击"确定"提交
4. 机构可在驳回原因修改后重新提交

### 3.3 机构信息查询

#### 3.3.1 机构列表

1. 进入"机构管理"→"机构列表"
2. 可按以下条件筛选：
   - 机构名称（模糊搜索）
   - 机构类型（民办/公办/公建民营）
   - 所属区域
   - 机构状态
3. 点击"查询"按钮

#### 3.3.2 机构详情

点击机构名称查看详情，包含：
- 基本信息
- 资质信息
- 收费标准
- 床位使用情况
- 在院老人数量
- 账户资金余额

### 3.4 机构解除监管审批

当机构申请解除监管时，需要进行审批。

#### 3.4.1 查看解除监管申请

1. 进入"机构管理"→"解除监管审批"
2. 查看待审批的申请列表

#### 3.4.2 审批流程

1. 点击申请编号查看详情
2. 核实以下信息：
   - 所有老人已退住
   - 账户余额已清零
   - 无未处理的预警
   - 资金划拨已完成
3. 点击"审批通过"或"审批驳回"
4. 填写审批意见并提交

### 3.5 机构评级管理

#### 3.5.1 机构评级

1. 进入"机构管理"→"机构评级"
2. 选择要评级的机构
3. 填写评级信息：
   - 评级星级（1-5星）
   - 评级时间
   - 评级说明
4. 点击"保存"提交

#### 3.5.2 评级公示

1. 进入"机构管理"→"评级公示"
2. 选择要公示的评级
3. 设置公示时间
4. 点击"发布公示"

### 3.6 黑名单管理

#### 3.6.1 加入黑名单

对于违规的机构，可以加入黑名单：

1. 进入"机构管理"→"黑名单管理"
2. 点击"添加黑名单"
3. 选择机构
4. 填写加入原因
5. 选择黑名单类型（临时/永久）
6. 点击"确定"提交

#### 3.6.2 移出黑名单

1. 在黑名单列表中找到机构
2. 点击"移出黑名单"
3. 填写移出原因
4. 点击"确定"提交

## 第四章 预警管理

### 4.1 预警信息查看

#### 4.1.1 预警列表

1. 进入"预警管理"→"预警列表"
2. 系统显示所有预警信息
3. 可按以下条件筛选：
   - 预警类型
   - 预警级别（高/中/低）
   - 处理状态
   - 发生时间

#### 4.1.2 预警类型说明

| 预警类型 | 触发条件 | 处理建议 |
|---------|---------|---------|
| 预收费用超额 | 入住人预收费用超过月费用12倍 | 核实费用合理性，提醒机构退款 |
| 押金超额 | 入住人押金超过床位费用12倍 | 核实押金合理性，提醒机构退款 |
| 入住人数超额 | 在住人数超过备案床位数 | 提醒机构控制入住，增加床位 |
| 预收总额超额 | 监管账户余额超出固定资产净额 | 联系机构划拨资金 |
| 风险保证金超低 | 账户余额接近风险保证金最低比例 | 提醒机构充值 |
| 大额支付 | 单笔或累计大额交易 | 核实交易真实性 |
| 交易对方高风险 | 交易对方在黑名单或可疑名单 | 暂停交易并核实 |

### 4.2 预警处理流程

#### 4.2.1 查看预警详情

1. 在预警列表中点击预警编号
2. 查看预警详细信息：
   - 预警机构
   - 预警类型
   - 预警描述
   - 触发阈值
   - 当前数值
   - 发生时间

#### 4.2.2 处理预警

1. 确认预警详情后，点击"处理预警"
2. 填写处理结果：
   - 处理方式（电话通知/现场核查/系统处理）
   - 处理说明
   - 后续跟进计划
3. 点击"提交"完成处理
4. 预警状态变更为"已处理"

#### 4.2.3 批量处理

对于同类型的预警，可以批量处理：

1. 勾选多个预警
2. 点击"批量处理"
3. 统一填写处理说明
4. 点击"提交"完成

### 4.3 预警规则配置

#### 4.3.1 查看预警规则

1. 进入"预警管理"→"预警规则配置"
2. 查看所有预警规则
3. 可按规则类型筛选

#### 4.3.2 修改预警规则

1. 点击规则后的"编辑"按钮
2. 修改以下参数：
   - 触发阈值
   - 预警级别
   - 启用状态
3. 点击"保存"提交

## 第五章 资金监管

### 5.1 监管账户查询

#### 5.1.1 账户列表

1. 进入"资金监管"→"监管账户"
2. 查看所有机构的监管账户信息
3. 可按机构名称筛选

#### 5.1.2 账户详情

点击机构名称查看账户详情：
- 监管账户余额
- 基本账户余额
- 预收服务费总额
- 押金总额
- 会员费总额

### 5.2 资金划拨审批

#### 5.2.1 查看待审批划拨

1. 进入"资金监管"→"资金划拨审批"
2. 查看状态为"待审批"的划拨申请

#### 5.2.2 审批划拨

1. 点击划拨单号查看详情
2. 核实划拨信息：
   - 划拨机构
   - 划拨金额
   - 划拨类型
   - 划拨期间
3. 确认无误后点击"审批通过"
4. 如有问题，点击"审批驳回"并说明原因

### 5.3 押金使用审批

#### 5.3.1 查看待审批申请

1. 进入"资金监管"→"押金使用审批"
2. 查看状态为"待监管审批"的申请

#### 5.3.2 审批流程

1. 点击申请编号查看详情
2. 核实以下信息：
   - 申请金额
   - 使用事由
   - 相关证明材料
   - 家属审批意见
3. 确认无误后点击"审批通过"
4. 系统自动执行：
   - 扣除老人账户押金余额
   - 生成划拨流水
   - 记录费用明细

### 5.4 银行对账

#### 5.4.1 监管账户流水

1. 进入"资金监管"→"银行对账"
2. 选择"监管账户流水"
3. 选择时间范围查询
4. 可下载电子回单

#### 5.4.2 收单交易流水

1. 选择"收单交易流水"标签
2. 查看所有支付交易记录
3. 可按支付渠道、交易时间筛选

## 第七章 常见问题

### 7.1 登录问题

**Q: 忘记登录密码怎么办？**

A: 联系系统管理员重置密码，或使用"忘记密码"功能通过注册邮箱找回。

**Q: 登录后提示"无权限访问"？**

A: 当前账号未分配相应权限，请联系系统管理员分配角色。

### 7.2 机构管��问题

**Q: 机构审批通过后如何通知机构？**

A: 系统会自动发送短信通知机构联系人，通知内容包含登录账号和密码。

**Q: 机构信息如何修改？**

A: 机构可以在"机构管理"端自行修改基本信息，修改后需重新审批。如需协助修改，可在机构列表中点击"编辑"按钮。

### 7.3 预警问题

**Q: 预警信息多久更新一次？**

A: 预警信息实时更新，系统每5分钟自动检测一次预警条件。

**Q: 如何关闭某类预警？**

A: 进入"预警规则配置"，找到对应规则，将启用状态改为"停用"即可。

### 7.4 资金问题

**Q: 监管账户余额与实际银行账户不一致？**

A: 可能存在未到账的交易，请在"银行对账"中核对交易流水，或联系技术支持处理。

**Q: 资金划拨失败如何处理？**

A: 查看"资金划拨记录"中的失败原因，常见原因包括：余额不足、银行账户信息错误、网络问题等。核实原因后可重新发起划拨。
