# 轮播图图片说明

## 图片尺寸要求

- **推荐尺寸**: 750px × 300px (宽×高)
- **格式**: JPG 或 PNG
- **文件大小**: 建议 < 500KB

## 文件命名

请将准备好的轮播图放置在此目录下,命名如下:

- `banner1.jpg` - 第一张轮播图(养老院大厅/外景)
- `banner2.jpg` - 第二张轮播图(康复训练区/活动室)
- `banner3.jpg` - 第三张轮播图(老人活动照片)

## 示例图片内容

1. **banner1.jpg**: 养老院大厅或外景照片
2. **banner2.jpg**: 老人活动或康复训练照片
3. **banner3.jpg**: 养老院环境或服务场景

## 如何替换

1. 准备好符合尺寸要求的图片
2. 将图片重命名为 banner1.jpg, banner2.jpg, banner3.jpg
3. 复制到当前目录 (`ruoyi-h5/public/images/banners/`)
4. 刷新浏览器即可看到新图片

## 注意事项

- 图片比例建议 5:2 (宽:高),避免变形
- 文件名严格按照上述命名,否则无法正确显示
- 如需添加更多轮播图,可修改 `src/views/home/index.vue` 中的 banners 配置
