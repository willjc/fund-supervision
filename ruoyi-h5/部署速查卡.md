# H5端部署速查卡

## 快速部署(3步)

### 1️⃣ 本地构建
```bash
cd ruoyi-h5
npm run build
```
或者使用脚本:
- Windows: 双击 `deploy.bat`
- Linux/Mac: `./deploy.sh`

### 2️⃣ 上传文件
```bash
scp -r dist/* root@服务器IP:/var/www/h5/
```

### 3️⃣ 配置Nginx
```nginx
server {
    listen 80;
    server_name h5.dayushaiwang.com;
    root /var/www/h5;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

重启Nginx:
```bash
nginx -t && systemctl restart nginx
```

## 常见问题

### ❌ 问题1: 页面刷新404
**解决**: Nginx添加
```nginx
try_files $uri $uri/ /index.html;
```

### ❌ 问题2: API请求跨域
**解决**: 后端配置CORS或使用Nginx代理

### ❌ 问题3: 静态资源404
**解决**: 检查Nginx root路径是否正确
