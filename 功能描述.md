# 养老机构预收费资金监管平台功能描述

## 项目概述

养老机构预收费资金监管平台是一个基于若依框架的综合性监管系统，旨在对养老机构的预收费资金进行全流程监管，保障老年人权益，规范养老机构收费行为，防范资金风险。平台采用"政府监管部门+养老机构+小程序端"的三端协同架构，实现资金监管、机构管理、风险预警、数据统计等核心功能。

## 系统架构

### 端点架构
- **民政监管部门端**：政府监管人员使用，负责机构准入、资金监管、风险预警
- **养老机构端**：养老机构使用，负责机构运营、入住管理、资金申请
- **小程序端**：老年人及家属使用，提供机构查询、费用缴纳、投诉建议等功能
- **数据统计平台**：大屏展示，提供全局数据分析和可视化

### 技术架构
- **后端**：Spring Boot + MyBatis + Spring Security + JWT
- **前端**：Vue 2.6 + Element UI
- **数据库**：MySQL + Redis
- **支付集成**：银行收单系统、支付网关
- **部署环境**：政务云部署
