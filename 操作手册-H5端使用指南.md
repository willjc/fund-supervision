## 第一章 快速入门

### 1.1 系统简介

养老机构预收费资金监管平台H5端是面向老人家属的移动端服务平台，主要功能包括：
- 养老机构查询与预约
- 在线缴费与订单管理
- 费用余额查询与明细
- 押金使用申请
- 机构评价与投诉建议

### 1.2 系统访问

**访问地址**: http://服务器地址/h5

**登录方式**:
- 微信授权登录
- 手机号验证码登录

### 1.3 账号���册登录

#### 1.3.1 微信登录

1. 打开微信扫描二维码
2. 点击"微信授权登录"
3. 首次登录需绑定手机号
4. 自动完成注册并登录

#### 1.3.2 手机号登录

1. 点击"手机号登录"
2. 输入手机号码
3. 点击"获取验证码"
4. 输入收到的验证码
5. 点击"登录"按钮

### 1.4 老人信息绑定

登录后需要绑定老人信息才能使用缴费、查询等功能：

1. 点击"我的"→"老人管理"
2. 点击"添加老人"
3. 填写老人信息：
   - 老人姓名
   - 身份证号
   - 与老人的关系
   - 联系电话
4. 点击"提交"完成绑定

## 第二章 机构查询

### 2.1 机构列表

#### 2.1.1 查找机构

1. 进入"首页"→"养老机构"
2. 系统展示附近机构列表
3. 可按以下条件筛选：
   - 区域位置
   - 机构类型（民办/公办/公建民营）
   - 机构评级
   - 价格区间

#### 2.1.2 机构列表信息

每个机构卡片展示：
- 机构名称
- 机构地址
- 评级星级
- 收费区间
- 距离距离
- 空置床位数

### 2.2 机构详情

#### 2.2.1 查看机构详情

1. 在机构列表中点击机构名称
2. 进入机构详情页
3. 查看以下信息：

**基本信息**:
- 机构照片
- 机构简介
- 服务特色
- 联系方式

**服务信息**:
- 床位数量
- 空置床位
- 收费标准

**设施展示**:
- 机构环境照片
- 公共设施
- 活动场所

### 2.3 收费标准查询

在机构详情页中，点击"收费标准"查看：
- 床位费用
- 护理费用
- 餐费标准
- 会员费
- 押金标准

### 2.4 机构收藏

#### 2.4.1 收藏机构

1. 在机构详情页点击"收藏"按钮
2. 机构加入收藏列表

#### 2.4.2 查看收藏

1. 进入"我的"→"我的收藏"
2. 查看已收藏的机构列表
3. 可取消收藏

### 2.5 机构预约

#### 2.5.1 预约参观

1. 在机构详情页点击"预约参观"
2. 填写预约信息：
   - 预约时间
   - 参观人数
   - 联系方式
   - 备注说明
3. 点击"提交预约"

#### 2.5.2 查看预约记录

1. 进入"我的"→"预约记录"
2. 查看预约状态和详情

## 第三章 在线缴费

### 3.1 选择服务类型

#### 3.1.1 选择老人

1. 进入"首页"→"在线缴费"
2. 选择要缴费的老人
3. 如未绑定老人，请先在"个人中心"中添加老人

#### 3.1.2 选择缴费类型

| 缴费类型 | 说明 |
|---------|------|
| 新入驻 | 首次入住养老机构 |
| 续费 | 续缴服务费用 |

### 3.2 确认订单

#### 3.2.1 填写订单信息

1. 选择服务期限（月数）
2. 选择床位类型（如有多个可选）
3. 选择护理等级
4. 系统自动计算费用明细

#### 3.2.2 费用明细

| 费用项目 | 说明 |
|---------|------|
| 床位费 | 每月床位费用 |
| 护理费 | 每月护理费用 |
| 餐费 | 每月餐费 |
| 会员费 | 一次性会员费 |
| 押金 | 可退还押金 |
| 订单总额 | 以上项目合计 |

#### 3.2.3 提交订单

1. 确认订单信息和金额无误
2. 点击"提交订单"按钮
3. 订单状态为"审核通过待付款"

### 3.3 在线支付

#### 3.3.1 发起支付

1. 在订单列表中找到待支付订单
2. 点击"立即支付"按钮
3. 确认支付金额和订单号
4. 点击"确认支付"

#### 3.3.2 完成支付

1. 系统跳转到支付页面
2. 模拟支付流程（演示环境）
3. 支付成功后自动返回
4. 订单状态更新为"已支付"

#### 3.3.3 支付后说明

支付成功后，系统会自动：
- 创建老人账户（如不存在）
- 根据订单金额分配账户余额
- 扣除首月服务费（入驻订单）
- 生成后续月份资金划拨计划

### 3.4 查看订单

#### 3.4.1 订单列表

1. 进入"我的"→"我的订单"
2. 查看所有订单
3. 可按订单状态筛选：
   - 全部
   - 待付款
   - 已支付
   - 已取消
   - 已退款

#### 3.4.2 订单详情

点击订单查看详情：
- 订单基本信息
- 费用明细
- 支付记录
- 服务期限

### 3.5 订单续费

#### 3.5.1 创建续费订单

1. 进入"我的"→"我的订单"
2. 找到已支付的订单
3. 点击"续费"按钮
4. 选择续费月数
5. 确认费用明细
6. 提交订单并支付

### 3.6 订单取消

#### 3.6.1 取消待支付订单

1. 在订单列表中找到待支付订单
2. 点击"取消订单"
3. 选择取消原因
4. 点击"确认取消"

## 第五章 押金管理

### 5.1 申请使用押金

#### 5.1.1 创建申请

1. 进入"我的"→"押金管理"
2. 选择老人
3. 点击"申请使用押金"
4. 填写申请信息：
   - 申请金额
   - 使用事由
   - 详细说明
   - 紧急程度
5. 上传相关证明材料（图片）
6. 点击"提交申请"

#### 5.1.2 注意事项

- 押金余额只能用于特定用途，如购买物品、医疗服务等
- 申请需要家属确认和监管部门审批
- 审批通过后资金直接划拨给养老机构

### 5.2 查看申请进度

#### 5.2.1 申请列表

1. 进入"我的"→"押金管理"
2. 查看所有押金使用申请
3. 显示申请状态和进度

#### 5.2.2 申请状态说明

| 状态 | 说明 | 所需时间 |
|------|------|---------|
| 草稿 | 暂存未提交 | - |
| 待家属审批 | 等待家属确认 | 1天内 |
| 家属已审批 | 家属已同意，待监管审批 | 2-3个工作日 |
| 待监管审批 | 等待民政部门审批 | 2-3个工作日 |
| 已通过 | 审批通过，资金已划拨 | - |
| 已驳回 | 审批未通过 | - |

#### 5.2.3 撤回申请

对于"待家属审批"状态的申请，可以撤回：

1. 在申请列表中找到申请
2. 点击"撤回申请"
3. 确认撤回操作

## 第七章 常见问题

### 7.1 登录注册问题

**Q: 忘记密码怎么办？**

A: 点击登录页面的"忘记密码"，通过手机号验证码重置密码。

**Q: 无法收到验证码怎么办？**

A: 请检查手机号码是否正确，确认手机信号良好，60秒后可重新获取验证码。

**Q: 微信登录失败怎么办？**

A: 请确保已关注公众号或授权应用正常，可尝试切换到手机号登录。

### 7.2 老人绑定问题

**Q: 可以绑定多个老人吗？**

A: 可以，一个账号可以绑定多个老人信息。

**Q: 如何解除老人绑定？**

A: 进入"我的"→"老人管理"，找到对应老人，点击"解除绑定"即可。

**Q: 绑定老人时提示"老人不存在"？**

A: 请确认老人信息已在养老机构系统中登记，且身份证号填写正确。

### 7.3 缴费问题

**Q: 支付成功后订单状态未更新？**

A: 请稍等1-2分钟刷新页面，如长时间未更新请联系客服。

**Q: 支付到一半取消怎么办？**

A: 订单会保持"待支付"状态，可以重新进入订单完成支付。

**Q: 如何申请退款？**

A: 已支付的订单需要退款，请联系养老机构或民政部门处理。

**Q: 缴费后多久能入住？**

A: 请与养老机构确认具体入住时间，订单支付后请联系机构安排入住。

### 7.4 费用查询问题

**Q: 余额为什么变少了？**

A: 服务费余额会按月自动划拨给养老机构，请查看费用明细了解具体扣款情况。

**Q: 押金为什么不能直接使用？**

A: 为保障资金安全，押金只能通过申请方式使用，确保用于老人相关支出。

**Q: 如何查看每个月的费用明细？**

A: 进入"费用查询"，按月份筛选查看详细费用记录。

### 7.5 押金申请问题

**Q: 押金申请多久能批下来？**

A: 正常流程需要3-5个工作日，包括家属确认和监管部门审批。

**Q: 押金申请被驳回怎么办？**

A: 查看驳回原因，修改后可重新提交申请。

**Q: 押金可以用来支付服务费吗？**

A: 不可以，押金只能用于购买物品、医疗服务等特定用途，不能支付服务费。

### 7.6 其他问题

**Q: 如何联系客服？**

A: 进入"我的"→"帮助中心"，查看客服联系方式。

**Q: 系统支持哪些浏览器？**

A: 建议使用微信、Safari、Chrome等主流浏览器访问。

**Q: 如何退出登录？**

A: 进入"我的"→"设置"，点击"退出登录"。
