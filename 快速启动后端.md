## 🚀 启动步骤

### 第一步: 重新加载Java项目

在VSCode中:

1. 按 `Ctrl + Shift + P` 打开命令面板
2. 输入: `Java: Clean Java Language Server Workspace`
3. 选择并回车执行
4. 在弹出的提示中选择 `Reload and delete`
5. 等待VSCode重新加载项目 (可能需要1-2分钟)

### 第二步: 打开启动类

1. 打开文件: `ruoyi-admin/src/main/java/com/ruoyi/RuoYiApplication.java`
2. 现在应该能看到 `Run | Debug` 按钮在 `main` 方法上方

### 第三步: 启动后端

**方式一: 点击Run按钮**
- 在 `public static void main` 上方点击 `Run` 按钮

**方式二: 使用快捷键**
- 打开 `RuoYiApplication.java` 文件
- 按 `F5` 启动

**方式三: 使用命令行**
```bash
cd D:\newhm\newzijin
mvn spring-boot:run -pl ruoyi-admin
```

## ✅ 启动前检查

### 1. MySQL服务必须启动

检查MySQL:
```bash
net start | findstr MySQL
```

如果未启动,启动MySQL:
```bash

# 或者
net start MySQL
```

### 2. Redis服务建议启动

检查Redis:
```bash
tasklist | findstr redis
```

如果未启动,进入Redis目录:
```bash
cd C:\Redis
redis-server.exe redis.windows.conf
```

## 📋 常见问题

### 问题1: 端口8080被占用

**错误**: `Port 8080 was already in use`

**解决**:
```bash
# 查找占用端口的进程
netstat -ano | findstr :8080

# 记下PID,然后结束进程
taskkill /F /PID <进程号>
```

### 问题2: MySQL连接失败

**检查**:
1. MySQL服务是否启动
2. 数据库 `newzijin` 是否存在
3. 用户名密码是否正确 (root/123456)

**测试连接**:
```bash
mysql -u root -p
# 输入密码: 123456
SHOW DATABASES;
```

### 问题3: Redis连接失败

Redis不是必需的,但最好启动:
```bash
cd C:\Redis
redis-server.exe
```

## 🔄 其他启动方式

### Maven命令启动
```bash
cd D:\newhm\newzijin
mvn clean spring-boot:run -pl ruoyi-admin
```

### 打包后启动
```bash
# 打包
mvn clean package

# 运行
cd ruoyi-admin\target
java -jar ruoyi-admin.jar
```

### 创建启动脚本

创建 `start-backend.bat`:
```batch
@echo off
echo 正在启动若依后端...
cd /d D:\newhm\newzijin\ruoyi-admin
mvn spring-boot:run
pause
```

双击运行即可启动。

## 🌐 部署到服务器

如果要部署到服务器:

```bash
# 1. 打包
mvn clean package

# 2. 上传jar
scp ruoyi-admin/target/ruoyi-admin.jar root@服务器IP:/opt/ruoyi/

# 3. 服务器启动
nohup java -jar /opt/ruoyi/ruoyi-admin.jar > ruoyi.log 2>&1 &
```
