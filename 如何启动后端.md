# 如何启动后端 - 3种方法

## 🎯 推荐方法1: 使用启动脚本 (最简单)

### 步骤1: 确保MySQL已启动

打开命令行,执行:
```bash
net start MySQL80
```

### 步骤2: 双击启动脚本

双击运行: **启动后端.bat**

脚本会自动:
- ✅ 使用JDK 1.8
- ✅ 检查MySQL状态
- ✅ 检查Redis状态
- ✅ 启动若依后端

### 启动成功标志

看到这个ASCII图案就说明启动成功了:
```
(♥◠‿◠)ﾉﾞ  若依启动成功   ლ(´ڡ`ლ)ﾞ
```

访问测试: http://localhost:8080

## 🔧 方法2: 使用VSCode的F5启动

### 前置条件

1. 安装VSCode插件:
   - Extension Pack for Java
   - Spring Boot Extension Pack

2. 等待插件加载完成 (右下角会有提示)

### 启动步骤

1. **打开Run视图**:
   - 按 `Ctrl + Shift + D`
   - 或点击左侧的"运行和调试"图标

2. **选择启动配置**:
   - 在顶部下拉菜单选择: `启动若依后端 (RuoYiApplication)`

3. **启动**:
   - 点击绿色播放按钮
   - 或按 `F5`

### 如果看不到启动配置

我已经创建了 `.vscode/launch.json` 配置文件。

如果还是看不到:
1. 关闭VSCode
2. 重新打开项目
3. 等待Java扩展完全加载 (可能需要2-3分钟)

## ⚠️ 常见问题

### 问题1: 端口8080被占用

**错误**: `Port 8080 was already in use`

**解决**:
```bash
# 查找占用8080的进程
netstat -ano | findstr :8080

# 结束进程 (替换<PID>)
taskkill /F /PID <PID>
```

### 问题2: MySQL连接失败

**检查MySQL是否启动**:
```bash
net start | findstr MySQL
```

**如果未启动**:
```bash
net start MySQL80
```

**测试连接**:
```bash
mysql -u root -p
# 输入密码: 123456
```

### 问题3: Java版本不对

**查看当前Java版本**:
```bash
java -version
```

**应该显示**:
```
java version "1.8.0_xxx"
```

**如果不是1.8**,请使用启动脚本,它会自动设置正确的Java版本。

### 问题4: Maven未找到

**检查Maven**:
```bash
mvn -version
```

**如果未安装Maven**:
1. 下载: https://maven.apache.org/download.cgi
2. 解压到: `C:\apache-maven-3.8.6`
3. 添加到系统PATH: `C:\apache-maven-3.8.6\bin`

## 🔄 停止后端

### 方法1: 在命令行窗口
按 `Ctrl + C` 停止

### 方法2: 在VSCode
点击停止按钮或按 `Shift + F5`

### 方法3: 强制结束进程
```bash
# 查找Java进程
tasklist | findstr java

# 结束进程
taskkill /F /PID <进程号>
```
