# 养老机构预收费资金监管平台 - 已完成功能清单

## 项目概述

**养老机构预收费资金监管平台**，采用Spring Boot + Vue架构，实现四端协同（民政监管端 + 养老机构端 + H5小程序端 + 数据统计平台），全面保障老年人预收费资金安��。

**技术栈**: Spring Boot + Vue 2/3 + MySQL + Redis + Element UI + Vant UI
**架构模式**: 前后端分离，微服务化设计
