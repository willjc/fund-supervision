| 系统模块 | 技术架构 | 用户群体 | 访问方式 |

| PC管理端 | Vue 2 + Element UI | 养老机构工作人员、民政监管人员 | 浏览器(PC) |
| H5移动端 | Vue 2 + Vant | 老人、家属 | 微信/手机浏览器 |
| 后端服务 | Spring Boot + MyBatis | - | RESTful API |
| 数据库 | MySQL 5.7+ | - | - |
| 缓存 | Redis | - | - |

### 6.2 部署方式

| 项目 | 部署方式 | 访问路径 |
|------|---------|---------|
| 后端API | JAR包 / Docker | http://server:8080/api |
| PC管理端 | Nginx静态托管 | http://server/admin |
| H5移动端 | Nginx静态托管 | http://server/h5 |

### 6.3 Nginx路径配置示例

```nginx
# PC管理端
location /admin {
    alias /www/ruoyi-ui/dist;
    try_files $uri $uri/ /admin/index.html;
}

# H5移动端
location /h5 {
    alias /www/ruoyi-h5/dist;
    try_files $uri $uri/ /h5/index.html;
}

# 后端API
location /api {
    proxy_pass http://localhost:8080;
}
```

## 八、技术选型原则

### 8.1 PC端选型原则
✅ 复用若依框架成熟方案  
✅ Element UI提供丰富的表格、表单组件  
✅ 适合后台管理系统的复杂业务  

### 8.2 H5端选型原则
✅ Vant提供完整的移动端组件  
✅ 触屏优化、大按钮设计  
✅ 支持下拉刷新、无限滚动  
✅ 适配各种屏幕尺寸  
✅ 可快速扩展为微信小程序  

### 8.3 后端选型原则
✅ Spring Boot生态成熟稳定  
✅ MyBatis灵活的SQL控制  
✅ JWT无状态认证  
✅ Redis缓存提升性能
