## 🚀 快速开始

### 第一步: 创建Vue 2项目

```bash
cd D:\newhm\newzijin

# 使用Vue CLI创建项目
vue create ruoyi-h5
```

**手动选择配置**:
- ❯ Manually select features
- 勾选: Babel, Router, Vuex, CSS Pre-processors, Linter
- Vue版本选择: **2.x**
- Router history mode: **Yes**
- CSS预处理器: **Less**
- ESLint配置: **ESLint + Standard config**
- Lint on save: **Yes**
- 配置文件位置: **In dedicated config files**

### 第二步: 安装依赖

```bash
cd ruoyi-h5

# 安装Vant 2.x和核心库
npm install vant@2.12.54 axios@0.27.2 js-cookie@3.0.5 dayjs@1.11.10 --save

# 安装移动端适配插件
npm install postcss-pxtorem@5.1.1 lib-flexible@0.3.2 --save

# 安装开发工具
npm install vconsole@3.15.1 --save-dev
```

### 第三步: 创建目录结构

```bash
cd src

# 创建主要目录
mkdir -p api config mixins router/modules store/modules utils
mkdir -p views/{home,institution,order,user,account,todo,elder,notice,auth}
mkdir -p components/{TabBar,InstitutionCard,OrderCard,Empty}
mkdir -p assets/{images,styles}
```

### 第四步: 复制配置文件

将以下文件从本文档复制到对应位置:
1. vue.config.js (项目根目录)
2. .env.development (项目根目录)
3. .env.production (项目根目录)
4. 所有utils/下的工具文件
5. config/dict.js配置文件

## 🔌 API接口定义

### src/api/account.js (押金账户-核心)

```javascript
import request from '@/utils/request'

/**
 * 获取押金余额
 * @param {Number} elderId 老人ID
 */
export function getAccountBalance(elderId) {
  return request({
    url: `/h5/account/balance/${elderId}`,
    method: 'get'
  })
}

/**
 * 获取费用明细列表
 * @param {Number} elderId 老人ID
 * @param {Object} params 查询参数
 */
export function getAccountDetail(elderId, params) {
  return request({
    url: `/h5/account/detail/${elderId}`,
    method: 'get',
    params
  })
}

/**
 * 获取押金使用申请列表
 * @param {Number} elderId 老人ID
 * @param {Object} params 查询参数
 */
export function getDepositApplyList(elderId, params) {
  return request({
    url: `/h5/account/applyList/${elderId}`,
    method: 'get',
    params
  })
}

/**
 * 获取押金使用申请详情
 * @param {Number} applyId 申请ID
 */
export function getDepositApplyDetail(applyId) {
  return request({
    url: `/h5/account/applyDetail/${applyId}`,
    method: 'get'
  })
}

/**
 * 家属审批押金使用申请(核心接口)
 * @param {Object} data 审批数据
 * @param {Number} data.applyId 申请ID
 * @param {Boolean} data.approved 是否同意
 * @param {String} data.opinion 审批意见
 */
export function approveDepositApply(data) {
  return request({
    url: `/h5/account/approve/${data.applyId}`,
    method: 'post',
    data: {
      approved: data.approved,
      opinion: data.opinion
    }
  })
}
```

### src/api/auth.js (认证接口)

```javascript
import request from '@/utils/request'

/**
 * 用户登录
 * @param {Object} data 登录数据
 * @param {String} data.phone 手机号
 * @param {String} data.smsCode 验证码
 */
export function login(data) {
  return request({
    url: '/h5/auth/login',
    method: 'post',
    data
  })
}

/**
 * 发送短信验证码
 * @param {String} phone 手机号
 */
export function sendSms(phone) {
  return request({
    url: '/h5/auth/sendSms',
    method: 'post',
    data: { phone }
  })
}

/**
 * 获取当前用户信息
 */
export function getUserInfo() {
  return request({
    url: '/h5/auth/userInfo',
    method: 'get'
  })
}
```
