# H5机构列表筛选功能实现指南

## 一、功能概述

根据用户需求,H5端机构列表页面需要实现以下筛选维度:

1. **床位情况**: 全部机构 / 有空位机构 / 免费试住机构
2. **机构性质**: 民办 / 公办 / 公建民营
3. **收住类型**: 自理 / 半护理 / 全护理 / 失能 / 失智 (支持多选)
4. **医疗条件**: 内设医疗机构 / 与医疗机构合作 / 自营医疗机构 / 无医养结合
5. **机构星级**: 1-5星
6. **总费用**: 1000以下 / 1001-2000 / 2001-3000 / 3001-4000 / 4000以上
7. **区域筛选**: 郑州市12个区县

### 3. 可用API接口

#### 机构列表查询
```
GET /h5/institution/list
```

**支持的筛选参数**:
- `institutionName`: 机构名称(模糊查询)
- `districtCode`: 区县代码(精确匹配)
- `institutionNature`: 机构性质(1-民办,2-公办,3-公建民营)
- `careLevels`: 收住类型(逗号分隔,如"1,2,3")
- `medicalCondition`: 医疗条件(1-4)
- `ratingLevel`: 星级评分(1-5)
- `priceRangeMin`: 价格下限
- `priceRangeMax`: 价格上限
- `freeTrial`: 免费试住(0-否,1-是)
- `pageNum`: 页码
- `pageSize`: 每页条数

#### 字典数据查询
```
GET /system/dict/data/type/{dictType}
```

**可用的字典类型**:
- `pension_district` - 区域字典
- `pension_institution_nature` - 机构性质
- `pension_care_level` - 收住类型
- `pension_medical_condition` - 医疗条件
- `pension_rating_level` - 星级评分
- `pension_bed_status` - 床位情况(前端UI辅助)

#### 2.3 加载字典数据

```vue
<script>
export default {
  created() {
    this.loadDictData()
    this.loadInstitutionList()
  },

  methods: {
    // 加载所有字典数据
    async loadDictData() {
      try {
        // 并行加载所有字典
        const [districts, natures, careLevels, medicalConditions, ratings] = await Promise.all([
          getDictData('pension_district'),
          getDictData('pension_institution_nature'),
          getDictData('pension_care_level'),
          getDictData('pension_medical_condition'),
          getDictData('pension_rating_level')
        ])

        this.dictOptions = {
          districts: districts.data || [],
          natures: natures.data || [],
          careLevels: careLevels.data || [],
          medicalConditions: medicalConditions.data || [],
          ratings: ratings.data || []
        }
      } catch (error) {
        this.$toast.fail('加载筛选选项失败')
        console.error('加载字典失败:', error)
      }
    }
  }
}
```

#### 2.5 筛选逻辑实现

```vue
<script>
export default {
  computed: {
    // 是否有激活的筛选条件
    hasActiveFilters() {
      return this.filterParams.districtCode ||
             this.filterParams.institutionNature ||
             this.filterParams.careLevels.length > 0 ||
             this.filterParams.medicalCondition ||
             this.filterParams.ratingLevel ||
             this.selectedPriceIndex > 0 ||
             this.selectedBedStatus !== 'all'
    }
  },

  methods: {
    // 床位情况变化
    onBedStatusChange(value) {
      if (value === 'free') {
        this.filterParams.freeTrial = '1'
      } else {
        this.filterParams.freeTrial = ''
      }
      // 有空位筛选需要后端支持,暂时忽略
      this.onFilterChange()
    },

    // 价格区间变化
    onPriceChange(index) {
      const priceOption = this.priceOptions[index]
      this.filterParams.priceRangeMin = priceOption.minValue
      this.filterParams.priceRangeMax = priceOption.maxValue
      this.onFilterChange()
    },

    // 筛选条件变化
    onFilterChange() {
      this.filterParams.pageNum = 1
      this.institutionList = []
      this.finished = false
      this.loadInstitutionList()
    },

    // 加载机构列表
    async loadInstitutionList() {
      if (this.loading || this.finished) return

      this.loading = true
      try {
        // 处理收住类型多选参数
        const params = {
          ...this.filterParams,
          careLevels: this.filterParams.careLevels.join(',') // 转为逗号分隔字符串
        }

        const response = await getInstitutionList(params)
        const newList = response.rows || []

        this.institutionList = this.filterParams.pageNum === 1
          ? newList
          : [...this.institutionList, ...newList]

        this.total = response.total
        this.finished = this.institutionList.length >= this.total

        this.loading = false
      } catch (error) {
        this.$toast.fail('加载失败')
        this.loading = false
        console.error('加载机构列表失败:', error)
      }
    },

    // 加载更多
    loadMore() {
      if (!this.finished) {
        this.filterParams.pageNum++
        this.loadInstitutionList()
      }
    },

    // 获取字典标签
    getDistrictLabel(value) {
      const item = this.dictOptions.districts.find(d => d.dictValue === value)
      return item ? item.dictLabel : value
    },

    getNatureLabel(value) {
      const item = this.dictOptions.natures.find(n => n.dictValue === value)
      return item ? item.dictLabel : value
    },

    getCareLevelLabel(value) {
      const item = this.dictOptions.careLevels.find(c => c.dictValue === value)
      return item ? item.dictLabel : value
    },

    getMedicalLabel(value) {
      const item = this.dictOptions.medicalConditions.find(m => m.dictValue === value)
      return item ? item.dictLabel : value
    },

    getRatingLabel(value) {
      const item = this.dictOptions.ratings.find(r => r.dictValue === String(value))
      return item ? item.dictLabel : value + '星'
    },

    // 清除单个筛选条件
    clearFilter(key) {
      if (key === 'ratingLevel') {
        this.filterParams[key] = null
      } else {
        this.filterParams[key] = ''
      }
      this.onFilterChange()
    },

    // 移除收住类型
    removeCareLevel(level) {
      const index = this.filterParams.careLevels.indexOf(level)
      if (index > -1) {
        this.filterParams.careLevels.splice(index, 1)
        this.onFilterChange()
      }
    },

    // 清除价格筛选
    clearPrice() {
      this.selectedPriceIndex = 0
      this.filterParams.priceRangeMin = null
      this.filterParams.priceRangeMax = null
      this.onFilterChange()
    },

    // 清除床位状态
    clearBedStatus() {
      this.selectedBedStatus = 'all'
      this.filterParams.freeTrial = ''
      this.onFilterChange()
    },

    // 重置所有筛选
    resetAllFilters() {
      this.filterParams = {
        districtCode: '',
        institutionNature: '',
        careLevels: [],
        medicalCondition: '',
        ratingLevel: null,
        priceRangeMin: null,
        priceRangeMax: null,
        freeTrial: '',
        pageNum: 1,
        pageSize: 10
      }
      this.selectedPriceIndex = 0
      this.selectedBedStatus = 'all'
      this.onFilterChange()
    }
  }
}
</script>
```

## 五、注意事项

1. **收住类型多选**: 前端以数组形式管理,发送请求前转为逗号分隔字符串
2. **价格区间**: 查询时机构价格区间与筛选区间有交集即返回
3. **字典缓存**: 字典数据可考虑缓存到 localStorage,减少请求
4. **性能优化**: 使用防抖处理筛选变化,避免频繁请求
5. **空状态提示**: 无筛选结果时显示友好提示
6. **移动端适配**: 注意筛选面板在小屏幕上的体验
