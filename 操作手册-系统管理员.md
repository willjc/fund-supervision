## 第二章 用户管理

### 2.1 用户列表

#### 2.1.1 查看用户列表

1. 进入"系统管理"→"用户管理"
2. 查看所有系统用户
3. 可按以下条件筛选：
   - 用户名（模糊搜索）
   - 手机号码
   - 状态（正常/停用）
   - 所属部门

#### 2.1.2 用户列表信息

用户列表展示以下信息：
- 用户ID
- 用户名
- 姓名
- 部门
- 手机号码
- 状态
- 创建时间
- 最后登录时间

### 2.2 新增用户

#### 2.2.1 创建新用户

1. 进入"系统管理"→"用户管理"
2. 点击"新增"按钮
3. 填写用户信息：
   - 用户名（必填，唯一）
   - 用户昵称
   - 部门
   - 手机号码
   - 邮箱
   - 用户性别
   - 状态（正常/停用）
4. 设置用户角色
5. 点击"确定"保存

#### 2.2.2 初始密码

新用户默认密码为：`123456`，首次登录后需修改密码。

### 2.3 修改用户信息

#### 2.3.1 编辑用户

1. 在用户列表中找到用户
2. 点击"修改"按钮
3. 修改可变更的信息
4. 点击"确定"保存

#### 2.3.2 重置密码

1. 在用户列表中找到用户
2. 点击"重置密码"按钮
3. 确认重置操作
4. 用户密码重置为：`123456`

#### 2.3.3 分配角色

1. 在用户列表中找到用户
2. 点击"更多"→"分配角色"
3. 选择要分配的角色
4. 点击"确定"保存

### 2.4 删除用户

1. 在用户列表中找到用户
2. 点击"删除"按钮
3. 确认删除操作
4. 用户及其关联数据将被删除

## 第三章 机构账号管理

### 3.1 机构账号列表

#### 3.1.1 查看机构账号

1. 进入"系统管理"→"机构账号管理"
2. 查看所有养老机构账号
3. 可按以下条件筛选：
   - 机构名称
   - 账号状态
   - 所属区域

### 3.2 创建机构账号

#### 3.2.1 手动创建机构账号

1. 进入"系统管理"→"机构账号管理"
2. 点击"新增机构账号"
3. 填写机构信息：
   - 机构名称
   - 统一社会信用代码
   - 机构类型
   - 联系人
   - 联系电话
   - 机构地址
4. 设置登录账号信息：
   - 登录用户名（自动生成）
   - 登录密码（默认123456）
5. 分配角色权限
6. 点击"确定"保存

#### 3.2.2 机构审批创建

机构账号通常通过机构入驻审批流程自动创建：
1. 民政部门审批机构入驻申请
2. 审批通过后系统自动创建机构账号
3. 账号信息通过短信通知机构联系人

### 3.3 机构账号管理

#### 3.3.1 重置机构密码

1. 在机构账号列表中找到机构
2. 点击"重置密码"按钮
3. 确认重置操作
4. 新密码将通过短信通知机构

#### 3.3.2 停用/启用账号

1. 在机构账号列表中找到机构
2. 点击"停用"或"启用"按钮
3. 确认操作

#### 3.3.3 修改机构权限

1. 在机构账号列表中找到机构
2. 点击"分配角色"
3. 选择或取消角色权限
4. 点击"确定"保存

## 第四章 角色与权限管理

### 4.1 角色管理

#### 4.1.1 角色列表

1. 进入"系统管理"→"角色管理"
2. 查看所有系统角色
3. 可按角色状态筛选

#### 4.1.2 预置角色说明

| 角色名称 | 角色标识 | 说明 |
|---------|---------|------|
| 超级管理员 | admin | 拥有所有权限 |
| 民政监管人员 | supervision | 民政监管部门人员 |
| 机构管理员 | pension_admin | 养老机构管理员 |
| 机构操作人员 | pension_user | 养老机构普通操作人员 |

### 4.2 新增角色

#### 4.2.1 创建角色

1. 进入"系统管理"→"角色管理"
2. 点击"新增"按钮
3. 填写角色信息：
   - 角色名称
   - 角色权限字符
   - 显示顺序
   - 状态
   - 备注
4. 点击"确定"保存

#### 4.2.2 分配菜单权限

1. 在角色列表中找到角色
2. 点击"修改"按钮
3. 切换到"菜单权限"标签
4. 勾选角色可访问的菜单
5. 点击"确定"保存

#### 4.2.3 分配权限

1. 在角色列表中找到角色
2. 点击"数据权限"按钮
3. 设置数据范围：
   - 全部数据
   - 自定义数据
   - 本部门数据
   - 本部门及以下数据
   - 仅本人数据
4. 点击"确定"保存

### 4.3 菜单管理

#### 4.3.1 菜单树结构

1. 进入"系统管理"→"菜单管理"
2. 查看系统菜单树
3. 可按菜单名称搜索

#### 4.3.2 菜单类型

| 类型 | 说明 | 图标 |
|------|------|------|
| 目录 | 导航菜单，不含页面 | 文件夹 |
| 菜单 | 具体功能页面 | 菜单 |
| 按钮 | 页面操作按钮权限 | 按钮 |

#### 4.3.3 新增菜单

1. 进入"系统管理"→"菜单管理"
2. 点击"新增"按钮
3. 填写菜单信息：
   - 菜单名称
   - 父级菜单
   - 显示顺序
   - 菜单类型
   - 菜单图标
   - 路由地址
   - 权限标识
4. 点击"确定"保存

## 第八章 常见问题

### 8.1 用户管理问题

**Q: 如何批量导入用户？**

A: 系统支持Excel批量导入用户，在用户管理页面点击"导入"按钮，下载模板后填写用户信息并上传。

**Q: 用户忘记密码如何处理？**

A: 有两种方式：(1)用户使用"忘记密码"功能自助重置；(2)管理员在用户管理页面重置密码。

**Q: 如何设置用户数据权限？**

A: 通过角色管理中的"数据权限"设置，可控制用户能看到的数据范围（全部/本部门/仅本人等）。

### 8.2 机构账号问题

**Q: 机构入驻后自动创建账号吗？**

A: 是的，民政部门审批通过机构入驻申请后，系统会自动创建机构账号并通过短信通知。

**Q: 机构账号可以分配多个吗？**

A: 可以，一个机构可以有多个账号，不同账号可以分配不同的权限。

**Q: 如何修改机构的关联机构？**

A: 在"机构账号管理"中点击"关联机构"，可以修改账号关联的养老机构。

### 8.3 权限问题

**Q: 如何给用户分配特定菜单权限？**

A: 先创建角色并分配菜单权限，然后将用户分配到该角色即可。

**Q: 菜单权限不生效怎么办？**

A: 请检查：(1)角色是否分配了菜单权限；(2)用户是否分配了该角色；(3)清除浏览器缓存后重新登录。

**Q: 如何隐藏某些菜单？**

A: 在"菜单管理"中将菜单状态设置为"停用"即可。

### 8.4 系统监控问题

**Q: 如何查看系统异常登录记录？**

A: 在"登录日志"中选择状态为"失败"的记录，可以查看所有登录失败的尝试。

**Q: 如何查看某个用户的所有操作记录？**

A: 在"操作日志"中按操作人员筛选，输入用户名即可查看该用户的所有操作记录。

**Q: 定时任务执行失败怎么办？**

A: 在"定时任务"中查看任务执行日志，根据错误信息排查问题，常见原因包括：数据库连接失败、参数配置错误等。

### 8.5 系统维护问题

**Q: 如何清理系统缓存？**

A: Redis缓存会自动清理，如需手动清理，请联系技术支持或使用Redis命令行工具清理。

**Q: 如何备份数据？**

A: 系统会自动备份数据库，也可以手动导出业务数据。建议定期备份数据库和上传文件。

**Q: 系统日志占用空间过大怎么办？**

A: 定期清理过期日志，建议保留最近3-6个月的日志记录。
